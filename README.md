## 📊 Project Summaries

### 1. Categorical Data Modeling
This project dives into Generalized Linear Models (GLMs). It explores how to handle binary and count outcomes, with a specific focus on **handling overdispersion** in count data. 
* *Key Skills:* Logistic Regression, Multinomial Modeling, Negative Binomial regression, Model Diagnostics (AIC/BIC).

### 2. Educational Multilevel Modeling (INVALSI Case Study)
An applied research project analyzing the Italian national assessment data for 5th-grade students. It accounts for the nested structure of data (students nested within schools) to determine the impact of socio-economic status vs. institutional quality.
* *Key Skills:* Multilevel/Mixed-Effects Modeling, ICC calculations, Random Intercepts/Slopes, Cross-level Interaction effects.

### 3. GAM Literature Review (Research Synthesis)
A deep-dive technical research project on **Generalized Additive Models (GAMs)**. It evaluates the shift from rigid linear models to flexible, non-linear spline-based smoothing functions to better model real-world phenomena.
* *Key Skills:* Non-parametric smoothing, Basis Functions (Splines), Penalized Likelihood, Model Interpretability.
